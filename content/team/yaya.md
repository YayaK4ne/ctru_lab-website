### Bio

Hi! My name is Yaya Kane, and I am a Computer Engineering student at McGill University. I am currently working as a Student Research Assistant within the Faculty of Dental Medicine and Oral Health Sciences. 

My role focuses on full-stack web development and hardware-software interfacing. I enjoy leveraging modern technologies to design, engineer, and optimize digital tools and architectures that streamline the lab's technical initiatives and research workflows.

### Technical Skills & Focus Areas
* **Full-Stack Development:** Designing responsive, modern, and accessible web systems using tools like React, Next.js, TypeScript, Tailwind CSS, and the Hugo framework ecosystem.
* **Systems Interfacing:** Writing automation scripts, managing version control via Git workflows, and building robust hardware-software integrations using hardware description languages like VHDL.
* **Digital Infrastructure:** Configuring modular layouts and custom style patterns to establish clean, professional user interfaces for collaborative academic platforms.
